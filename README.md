# Qinmeiorg

此套主题主要是将 wordpress 改造成纯api的站点，以便实现前后端分离的技术栈，目前的进度已经大致完成，唯一的问题就是需要安装 JWT token 插件，具体的请查看 wiki 的安装说明。

**主要注意的是本主题仅为后台，前台主题请自行编写。**
